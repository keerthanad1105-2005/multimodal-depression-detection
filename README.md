# Multimodal Depression Detection System

## Models Used

- RoBERTa (Text)
- MobileViT (Face Emotion Recognition)
- Wav2Vec2 (Speech Emotion Recognition)

## Datasets

- Reddit Depression Dataset
- FER2013
- RAVDESS

## Tech Stack

- FastAPI
- PyTorch
- Transformers
- HTML
- CSS
- JavaScript

## Features

- Text-based depression detection
- Face emotion recognition
- Voice emotion recognition
- Multimodal fusion
- Web dashboard
